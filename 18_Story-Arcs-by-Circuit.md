# Story Arcs by Circuit (Missions Map)

This pack includes full arcs that work across all country routes.

## Shared Chapter Structure
- CH1 Tryouts + Choose Circuit + Assign Roles
- CH2 Funding + Facilities + Sponsor pressure
- CH3 Training Blocks (format-specific)
- CH4 Setback (conflict/burnout/logistics)
- CH5 Qualifiers
- CH6 Finals / Global push

## Circuits included
### Solo Circuit
- Local entry (build confidence)
- National final prep (pressure + precision)
- (Optional) Mystery box (adaptability test)

### Bocuse Track
- Candidate + commis pairing
- Theatre rehearsals (platter + plates)
- Trials → Final

### Team Global Track (World Cup/Olympics)
- Selection camp + station map
- Workflow rehearsals + relay/handoffs
- Qualifier → Olympics

### Pastry Track
- Showpiece discipline (structure + finish)
- Salon display (symmetry + perfection)

## Design note
Missions unlock via flags and set up competitions by:
- boosting readiness and key skills
- adding or reducing risk
- setting pre-competition flags that influence penalty chances
