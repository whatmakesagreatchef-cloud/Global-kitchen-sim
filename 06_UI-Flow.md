# UI Flow

## Screens
1) Main Menu
- New Career, Continue, Import/Export Save, Settings

2) Route Select
- Country cards with:
  - pressures
  - learning outcomes
  - modifiers preview
  - conflict style preview

3) Draft Team
- Pick 3–5 archetypes
- Show predicted cohesion/conflict risk

4) Week Hub
- Week briefing + upcoming gates
- Choose missions (1–2)

5) Mission Screen
- Story text
- Choices
- Outcome summary (applied effects)

6) Event Screen (if triggered)
- Event text + response choice

7) Competition Day (on comp weeks)
- Show readiness + rubric
- Apply penalties if flags/events triggered
- Score + feedback

8) Reports
- Readiness trend
- Skill gaps
- Cohesion/relationships
- Budget forecast
- “What you learned” tags

9) Season End
- Route summary + unlocks
