# Gameplay Loop

## New Career
1) Choose a country route
2) Draft starting team (archetype picks)
3) Begin Week 1 mission

## Weekly cycle
1) Read week briefing (constraints, upcoming gates)
2) Choose 1â€“2 missions (story/training/funding)
3) Resolve checks (stat + modifiers + seeded RNG)
4) Apply effects (skills, fatigue, morale, budget, relationships, flags)
5) Trigger one event (conditional first, else random roll)
6) If competition week: run scoring and penalties
7) End-of-week summary: warnings + recommended actions

## Pacing rules
- The player should always feel:
  - one pressure (time/budget)
  - one people problem (conflict)
  - one performance target (skill gap)
